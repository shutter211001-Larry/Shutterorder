## 🔬 技術深度掃描驗證 (Deep Code Scan Evidence)
*為確保行銷企劃的真實性與技術落地能力，本企劃基於對底層程式碼的絕對掃描：*
* **資料庫層 (`schema.prisma`) 掃描**：系統具備高達 40+ 個關聯模型。除了基礎的 `User`, `Customer`, `Order`, `MenuItem`，更包含了 `StaffAttendance`, `AttendanceCorrectionRequest`, `ShiftRequirement`, `GroupOrderSession`, `CookieConsent`, `JobRole` 等高度複雜的企業級模型。
* **後端路由 (`api-server/src/routes`) 掃描**：共計 27 支核心路由，涵蓋從基礎的點餐 (`order.routes`) 到高階的請假 (`leave.routes`)、考勤 (`attendance.routes`)、自動化行銷 (`automation.routes`) 與發票開立 (`invoice.routes`)。
* **前端管理後台 (`adminfront/src/pages`) 掃描**：具備高達 71 個功能頁面。包含極其專業的 `AIMenuDetection.tsx` (AI 菜單辨識)、`CounterDisplay.tsx` (櫃檯叫號機)、`KitchenDisplay.tsx` (KDS 廚房顯示)、`ShiftRequirementsPage.tsx` (排班需求管理)、`AttendancePayroll.tsx` (考勤薪資計算) 以及 `SettingsFranchise.tsx` (加盟店設定)。
* **前端消費者端 (`storefront/src/pages`) 掃描**：包含 `Checkout.tsx`, `LinePayConfirm.tsx`, `OrderHistory.tsx`, `OrderStatus.tsx` 等完整消費者旅程。

## 一、 系統完整功能解析 (Full Feature Scan)

基於上述程式碼掃描，我們萃取出以下企業級核心功能矩陣：

### 1. 多渠道點餐與結帳 (Omnichannel Ordering & Payments)
* **全方位點餐模式**：內用 (Table)、外送 (Delivery)、外帶自取 (Pickup) 與預約訂餐 (`OrderCreate.tsx`)。
* **同桌多人點餐系統**：支援掃碼入座與 PIN 碼驗證，允許同一桌 (`tableId`) 多位顧客加入同一個訂單 Session 進行共同點餐與個別結算 (`group-order.routes`)。
* **智慧結帳與在地化金流**：支援 Stripe 信用卡、LINE Pay (`LinePayConfirm.tsx`) 與現金結帳。並整合台灣在地電子發票開立 (`SettingsInvoice.tsx`)。
* **硬體生態串接**：透過 Web Bluetooth API (`bluetoothPrinter.ts`) 支援網頁端直接連接熱感應印表機，以及專屬的櫃台叫號顯示介面 (`CounterDisplay.tsx`)。

### 2. 多門店與靈活菜單管理 (Menu & Franchise Ops)
* **加盟與多門店架構**：單一後台管理多品牌/多分店 (`SettingsFranchise.tsx`, `LocationForm.tsx`)，獨立設定營業時間、配送費 (`DeliveryZoneList.tsx`)。
* **高階智能菜單**：無限層級分類、複雜客製化選項，並導入了創新的 **AI 菜單辨識建檔 (`AIMenuDetection.tsx`)**。
* **庫存與廚房看板**：具備即時庫存追蹤 (`StockManagement.tsx`)，以及支援 Kanban 拖曳與 Socket.io 即時更新的廚房看板 (`KitchenDisplay.tsx`)。

### 3. CRM 會員經營與行銷增長 (Marketing & Loyalty)
* **全通路會員制**：跨店會員資料整合、忠誠度點數累積 (`CustomerLoyalty.tsx`)。
* **多樣化促銷模組**：支援折扣碼 (`CouponForm.tsx`) 與自動化行銷腳本 (`AutomationRuleForm.tsx`)。
* **LINE 深度生態系整合**：LINE OA 綁定、LINE 內點餐與推播 (`SettingsLine.tsx`)。

### 4. 企業級人事與排班 ERP (HR & Staff Management)
* **考勤與薪資**：支援手機 QR 碼打卡 (`AttendanceQRGenerator.tsx`)、遲到早退記錄、考勤校正審批 (`AttendanceCorrections.tsx`) 與薪資試算 (`AttendancePayroll.tsx`)。
* **智慧排班系統**：員工線上排班 (`RosterManagement.tsx`)、人力需求預估 (`ShiftRequirementsPage.tsx`) 與請假簽核 (`LeaveApprovals.tsx`)。
* **職務權限控制**：細粒度的角色權限設定 (`JobRoleSettings.tsx`)。

### 5. 商業智慧與系統底層 (Analytics & Infrastructure)
* **營收儀表板與財務**：即時營收圖表與進階財務報表 (`Finance.tsx`, `Dashboard.tsx`)。
* **國際化與法規合規**：13 國語系支援、Cookie 同意記錄 (`ConsentLog.tsx`)、操作稽核日誌 (`AuditLog.tsx`)。
* **開發者與 API 中心**：Webhook 串接與開發者效能監控 (`DeveloperMetrics.tsx`)。

## 二、 產品市場定位與痛點解決 (Market Positioning)

### 目標客群 (Target Audience)
1. **中大型連鎖/加盟餐飲品牌**：極需跨店數據整合、HR 排班與薪資結算功能。
2. **小型獨立商家 / 微型餐飲 (餐車、咖啡外帶店)**：免購買昂貴特規 POS 機，只要一台 iPad 加上藍牙出單機即可低成本開店。系統內建 10 款高質感佈景主題，幫助小店零成本打造媲美大品牌的專屬點餐網站。
3. **注重利潤的在地餐廳**：希望擺脫外送平台 30% 高抽成，透過 LINE Pay 與自有平台將過客轉化為熟客。

### 解決的市場痛點
* 🚫 **抽成過高**：自建點餐系統實現 **0% 訂單抽成**。
* 🚫 **軟體疊床架屋**：將 POS、線上點餐、HR 考勤、KDS 廚房看板完美融合成一體化系統，取代數十套軟體。
* 🚫 **高昂的硬體綁定費**：支援瀏覽器藍牙直連印表機，餐廳可使用自備的 iPad 或一般電腦。

## 三、 階段性行銷推廣策略 (Marketing Strategy)

### Phase 1: 開發者口碑與技術品牌建立
* **開源火力展示**：在 GitHub / Product Hunt 主打「內建 AI 菜單辨識與 HR 排班的終極餐飲 ERP」，吸引技術型餐飲創業者。
* **技術文章曝光**：發表「如何利用 Web Bluetooth 在瀏覽器實現出單」或「Socket.io KDS 實戰」。

### Phase 2: 商業概念驗證與 SaaS 營運
* **推出 Shutter Cloud 雲端代管**：多數老闆不懂架設。推出「免安裝、月費制」的 SaaS 服務，賺取穩定的訂閱費。
* **「免費數位健檢」行銷**：投放 FB/IG B2B 廣告，提供「利潤試算表」，並免費為前 50 名商家建立點餐網站。
* **短影音火力展示**：拍攝短影音展示「手機掃碼打卡 → 排班異常校正」如何幫店長省下每個月 20 小時的算薪水時間。

### Phase 3: 生態系擴張與企業方案
* **經銷商推薦計畫 (Affiliate)**：邀請 POS 硬體商、餐飲顧問成為代理商。
* **企業級白牌 App**：協助旗艦品牌打包專屬的 iOS/Android App (利用內建 `mobile` 模組)。
* **當地金流與發票 API 插件市集**：開放開發者生態系，收取串接授權費。
